# Telco Marktforschung

Dieses Repository stellt Vorlagen und Arbeitsanweisungen für die Marktforschung im Telekommunikationsbereich bereit.

## Inhalte

- `docs/research_catalog_template.csv`: Finale Tabellenstruktur zur Erfassung von Quellen (Quelle, Datum, Kategorie, Summary, Relevanz).
- `docs/workflow_guide.md`: Anleitung zur Überführung der Daten in Notion oder Airtable sowie zu den ersten Visualisierungen in Data Studio oder Tableau.

## Nutzung

1. Importiere die CSV-Datei in dein bevorzugtes Spreadsheet-Tool und erweitere sie mit weiteren Einträgen.
2. Folge der Workflow-Anleitung, um eine Notion- oder Airtable-Datenbank anzulegen, Tags zu vergeben und Dashboards aufzubauen.
3. Aktualisiere die Daten regelmäßig und erweitere die Visualisierungen nach Bedarf.
